## Confluence Integration
Export directly to your team’s Confluence wiki with automatic organization.

<AccordionGroup>
  <Accordion icon="copy" title="Quick Setup">
    1. Generate Atlassian API token from account settings 
    2. Configure PUG with your Confluence credentials
    3. Add your Confluence spaces
    4. Export documentation directly to wiki
  </Accordion>
    <Accordion icon="copy" title="Configuration on PUG">
    **Username:** Your Confluence email address

    **API Token:** Generated from Atlassian account

    **Confluence URL:** Your organization’s Confluence URL
  </Accordion>
</AccordionGroup>


## Notion Integration
Export to Notion databases for team collaboration and knowledge management.

<AccordionGroup>
  <Accordion icon="copy" title="Quick Setup">
  1. Create Notion integration at notion.so/my-integrations 
  2. Set up a documentation database in Notion
  3. Share database with your PUG integration
  4. Configure PUG with integration token
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title='Confluence Issues'>
  **Authentication failed:**
      - Verify email and API token are correct
      - Check API token hasn’t expired
      - Ensure write permissions to the space
    
  **Space not found:**
      - Verify space key is correct (case-sensitive)
      - Check space exists and you have access
      - Confirm Confluence URL is complete
  </Accordion>

  <Accordion title="Notion Issues">
  **Database not accessible:**
      - Verify database ID is correct
      - Check integration has database access
      - Ensure database is shared with PUG integration

  **Token invalid:**
      - Regenerate Notion integration token
      - Verify complete token was copied
      - Check integration hasn’t been revoked
  </Accordion>

  <Accordion title="General Issues">
  **No documentation found:**
      - Generate documentation first with PUG
      - Check .cursor/rules/ contains .mdc or files
      - Verify file paths in project
  
  **Export failed:**
      - Check internet connection
      - Verify write permissions to output directory
      - Try exporting individual files to isolate issue
  </Accordion>
</AccordionGroup>

## Best Practices
### For Teams 
- Use consistent naming for spaces and databases
- Set up templates for standardized formatting
- Regular exports to keep team documentation current
- Proper permissions in Confluence/Notion

### For Organizations
- Centralized management with shared spaces
- Brand consistency with custom styling
- Version control and access auditing
- Regular review of documentation access
